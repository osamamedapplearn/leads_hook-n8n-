كنا نحرق ساعات طويلة يومياً في البحث اليدوي على (فيسبوك، انستجرام، وتيك توك) للعثور على شركات، محلات، وعيادات تحتاج لخدماتنا. نجمع الأرقام، نؤلف رسائل مخصصة لكل عميل، ونرتبها في جداول.. كانت عملية مرهقة ومشتتة وتستنزف طاقتنا.

الفكرة: "نحن نبيع حلول الأتمتة والذكاء الاصطناعي للشركات.. لماذا لا نبني نظاماً ذكياً يقوم بهذا العمل بدلاً منا ويجلب لنا العملاء حتى باب الوكالة؟"

🤖 الحل: "موظف مبيعات رقمي" لا ينام

قمنا ببناء Workflow ذكي متكامل على منصة n8n، وظيفته قراءة طلبنا البسيط (مثال: "عايز 50 براند ملابس في مصر على تيك توك"). ومن ثم ينطلق ليمسح منصات التواصل الاجتماعي، يصطاد العملاء المحتملين، يقيم جودتهم، يكتب لكل واحد فيهم رسالة بيع (Cold Message) مخصصة، ويرصهم جاهزين في قاعدة البيانات.

🛠️ الترسانة التقنية (The Tech Stack)

الأداة

الوظيفة في النظام

n8n

المايسترو (Orchestrator) الذي يربط جميع الأنظمة ويدير مسار تدفق البيانات.

Google Vertex AI

العقل المدبر. يحلل الطلب، يوجه محركات البحث، ويكتب رسائل مبيعات جذابة باللهجة المصرية.

🔄 القرار الاستراتيجي: لماذا Apify وليس Meta Graph API؟

في البداية، فكرنا في استخدام الطريق الرسمي وهو Meta Graph API للبحث عن صفحات فيسبوك وانستجرام، لكن اكتشفنا أنه "فخ" للبحث عن عملاء جدد للأسباب التالية:

قيود صارمة: الـ API مصمم لإدارة صفحاتك أنت، وليس للبحث المفتوح عن بيانات شركات أخرى.

موافقات معقدة: يتطلب مراجعات طويلة وأذونات معقدة، مع خطر حظر الحساب بسهولة.

نقص البيانات: لا يوفر دائماً أرقام الهواتف أو الروابط المخفية في الـ Bio بسهولة.

💡 الحل السحري كان Apify:
أدوات Apify تعمل كـ "متصفح بشري سريع". تدخل للمنصات وتسحب البيانات العامة (Public Data). هذا وفر لنا:

تخطي كامل لقيود الـ APIs الرسمية.

استخراج أدق التفاصيل من الـ Bio (أرقام الهاتف، الإيميلات).

مرونة رهيبة في البحث الدلالي والهاشتاجات على تيك توك وانستجرام.

⚙️ كيف يعمل النظام من الداخل؟ (The Workflow)

بمجرد كتابة الطلب في الـ Chat Trigger، يمر النظام بـ 4 محطات في ثوانٍ معدودة:

🧠 الموجه الذكي (AI Router): يفهم الطلب ويولد "كلمة البحث" (Payload) التي تناسب خوارزمية كل منصة (لأن بنية بحث فيسبوك تختلف كلياً عن تيك توك).

🕵️‍♂️ الاستخراج (Data Extraction): يتم إرسال استعلام البحث إلى أدوات Apify لاصطياد الحسابات والصفحات المطابقة.

🔬 الفلترة والمبيعات (Process & Analyze): كود JavaScript ينظف البيانات، ثم يتدخل الذكاء الاصطناعي ليقيم جودة العميل ويستخرج رقمه، ويكتب له رسالة مبيعات مخصصة.

💾 التخزين (Storage): تُدرج البيانات منظمة في Google Sheets، لتنتظر فقط ضغطة "إرسال" من فريق المبيعات.

🎯 النتيجة النهائية

انتقلنا من مرحلة "البحث اليدوي العشوائي" الذي يستغرق أياماً، إلى امتلاك "ماكينة توليد ليدات" تعمل بضغطة زر وتوفر لنا مئات العملاء المستهدفين يومياً.


شيت الليدات النهائي: تحليل ذكي للعميل، استخراج للبيانات، وتوليد رسالة مبيعات مخصصة لكل نشاط تجاري.

🚀 كيفية تشغيل المشروع (Installation)

قم بتنصيب منصة n8n.

قم بإنشاء حساب على Apify واحصل على الـ API Token.

قم بإعداد مشروع على Google Cloud وتفعيل Vertex AI API والحصول على Service Account JSON.

قم بإنشاء Google Sheet وتجهيز الأعمدة المطلوبة (Business Name, Phone Number, Page URL, إلخ).

قم باستيراد ملف workflow.json المرفق في هذا المستودع إلى بيئة n8n الخاصة بك.

قم بإدخال الـ Credentials الخاصة بك في الـ Nodes.

اضغط Execute Workflow واستمتع بالصيد! 🎣
